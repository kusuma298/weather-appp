# Weather App

This is a simple weather application built using HTML, CSS, and JavaScript. It allows users to search for any city and view the current weather details by fetching real-time data from the OpenWeatherMap API.

## Features

- Search weather by city name
- View current temperature
- Check humidity and wind speed
- See the current weather condition
- Simple and responsive user interface
- Displays an error message for invalid city names

## Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeatherMap API
- Fetch API

## Project Structure

```
Weather-App/
│── index.html
│── style.css
│── script.js
```

## Getting Started

1. Download or clone this repository.
2. Get a free API key from OpenWeatherMap.
3. Open the `script.js` file and replace:

```javascript
const apiKey = "YOUR_API_KEY";
```

with your own API key.

4. Open `index.html` in your browser to run the application.

## About

I built this project to improve my JavaScript skills and learn how to work with REST APIs. It helped me understand how to fetch real-time data, handle asynchronous requests using the Fetch API, and update web pages dynamically based on user input.
